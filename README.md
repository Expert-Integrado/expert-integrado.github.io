# expert-integrado.github.io

Hub de páginas públicas dos projetos da Expert Integrado (GitHub Pages da org).

## Padrão

- Projeto com **repo público** hospeda a própria página no `docs/` dele (ex.: [expert-brain](https://expert-integrado.github.io/expert-brain/)).
- Projeto com **repo privado** ganha uma pasta aqui (`<projeto>/index.html`) e a página sai em `https://expert-integrado.github.io/<projeto>/` — mesma URL que teria se o Pages fosse do próprio repo.
- A raiz ([index.html](index.html)) é o índice de todos os projetos.

## Regras

- Conteúdo 100% público: nenhum segredo, token, ID de projeto, telefone ou dado pessoal entra aqui. Auditar antes de cada push.
- Página nova = pasta nova + card no `index.html` da raiz + link no README do repo do projeto.
